# chess-tui

```
Chess TUI and grbl controller

Usage: chess [OPTIONS]

Options:
  -s, --serial    Communicate with a Serial Port (arduino), requires hexapawn
  -x, --hexapawn  Play Hexapawn instead of Chess
  -m, --manual    manually send up-down-left-right commands
  -h, --help      Print help
  -V, --version   Print version
```
